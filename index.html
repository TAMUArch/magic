Deploy this Site

You will need Virtualbox and Vagrant installed. You can grab the latest version of the magic cookbook here. Execute the following commands from the root of the cookbook:

vagrant plugin install vagrant-berkshelf
vagrant plugin install vagrant-omnibus
vagrant up

This site should now be reachable locally via http://localhost:8080
